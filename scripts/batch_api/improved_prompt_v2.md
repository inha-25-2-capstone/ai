# 정치 뉴스 논조 분류 프롬프트 (v2 - 종합 판단 + 행위 목록)

## 최종 프롬프트

```
당신은 정치 뉴스 분석 전문가입니다. 뉴스 기사의 논조를 정확히 분석하세요.

다음 뉴스 기사를 읽고, 기사의 스탠스를 분류하세요.

제목: {title}
기사 내용: {content}

## 판단 기준

기사의 논조를 판단할 때는 **4가지 요소를 종합적으로 고려**하세요:

### 1. 행위의 성격 (동사 + 목적어)

**긍정적 행위 (옹호 신호):**
- 지원/복구: 예산 복구, 민생 지원, 피해 구제, 취약층 지원
- 추진/실천: 정책 추진, 개혁 실천, 약속 이행, 협력 요청
- 성과: 최초 달성, 성공적 진행, 효과적 대응, 신속한 조치
- 예: "예산 복구 지시", "민생 지원 추진", "개혁 약속 이행"

**부정적 행위 (비판 신호):**
- 탄압/억압: 언론 탄압, 권한 남용, 직권 남용, 사찰
- 불법/의혹: 사법농단, 범죄 의혹, 불법 행위, 규정 위반
- 회피/거부: 소환 불응, 수사 거부, 책임 회피
- 인사 문제: 보좌진 갈아치움, 인사 강행
- 예: "소환 불응", "직권 남용", "보좌진 갈아치움"

**중립적 행위:**
- 일상 활동: 회의 참석, 발표, 일정 소화, 방문
- 선거 활동: 지지 호소, 유세, 지원 유세
- 공식 행사: 기자회견, 추념식 참석
- 예: "지지 호소", "회의 참석", "발표"

### 2. 서술 방식 (같은 행위, 다른 표현)

**우호적 서술 (옹호):**
- 긍정 동사: 추진하다, 실천하다, 당부하다, 격려하다
- 신속성 강조: "빠른", "즉시", "적극적"
- 프레이밍: "개혁", "혁신", "쇄신"

**비판적 서술 (비판):**
- 부정 동사: 강행하다, 밀어붙이다, 갈아치우다
- 문제 강조: "논란", "의혹", "졸속"
- 프레이밍: "독재", "농단", "유린", "괴물"

**중립적 서술 (중립):**
- 단순 동사: 밝히다, 말하다, 발표하다, 참석하다
- 감정 없음: 평서형 서술, 사실 나열

### 3. 인용 vs 기자 서술

**핵심 원칙: 인용문의 내용이 아니라 기자가 어떻게 서술했는가?**

- 중립: "대통령이 '열심히 하겠다'고 **밝혔다**" → 단순 인용
- 옹호: "대통령이 민생 지원을 **약속했다**" → 긍정적 행위 강조
- 비판: "대통령이 논란 정책을 **강행했다**" → 부정적 서술

### 4. 전체 맥락과 강조점

**무엇을 부각시키는가?**

- 옹호: 성과, 신속한 대응, 피해자 구제, 긍정적 변화
- 비판: 문제점, 의혹, 불법성, 피해 발생
- 중립: 사실만 나열, 일정/행사 내용

**예시로 이해하기:**

같은 "예산" 사건을 다르게 서술:
- 옹호: "李 대통령, 삭감된 예산 **빠른 복구** 당부" (신속한 조치 강조)
- 비판: "李 정부, 尹 정책 **번복** 지시" (전 정부 부정 강조)
- 중립: "예산 복구 지시" (사실만)

## 종합 판단 프로세스

1. **행위 확인**: 무엇을 했는가? (동사+목적어)
   - 위 "행위 목록"에서 해당 행위가 긍정적/부정적/중립적인지 확인

2. **서술 방식 확인**: 어떻게 표현했는가?
   - 같은 행위라도 "추진" vs "강행"처럼 다른 동사 사용?
   - "빠른", "논란", "최초" 같은 수식어 사용?

3. **인용 vs 서술 구분**:
   - 단순히 발언을 인용만 했는가?
   - 아니면 기자가 행위를 평가/해석했는가?

4. **전체 어조**:
   - 읽는 사람이 받는 인상: 우호적? 비판적? 중립적?
   - 애매하면 제목의 첫인상으로 판단 (중립로 도피 금지!)

## 분류 예시

### 예시 1: 옹호 (0)
제목: 尹 정부 삭감한 광복회 예산 李 대통령, '빠른 원상복구' 당부
내용: 이재명 대통령은 6일 오전 서울 동작구 국립서울현충원에서 현충일 추념식을 마친 뒤 퇴장하는 과정에서 참석자가 삭감된 광복회 예산에 대해 안타까움을 토로하자 윤석열 정부가 삭감한 광복회 예산을 빠르게 원상 복구하라고 지시하였다.

분류: 0 (옹호)
판단 근거:
- 행위: "예산 원상복구 지시" → 긍정적 행위 (복구, 지원)
- 서술: "빠르게" → 신속성 강조, 우호적
- 전체: 이재명 대통령의 적극적 대응 부각

### 예시 2: 옹호 (0)
제목: 선관위원장 "선거결과 승복해주시길 정확하고 빈틈없이 개표할 것"
내용: 선거관리위원회 위원장이 대선을 앞두고 정확하고 빈틈없는 개표를 약속하며 모든 국민이 선거 결과에 승복해줄 것을 당부했다.

분류: 0 (옹호)
판단 근거:
- 행위: "정확한 개표 약속", "당부" → 긍정적 책임감
- 서술: "정확하고 빈틈없이" → 신뢰감 조성
- 전체: 선관위의 공정성 강조

### 예시 3: 중립 (1)
제목: 지지 호소하는 김문수 후보
내용: 김문수 국민의힘 대선 후보가 제21대 대통령 선거일을 하루 앞둔 2일 오후 부산역광장에서 유세를 갖고 지지를 호소했다.

분류: 1 (중립)
판단 근거:
- 행위: "지지 호소" → 중립적 선거 활동
- 서술: "호소했다" → 단순 서술, 평가 없음
- 전체: 사실만 전달, 감정 표현 전혀 없음

### 예시 4: 중립 (1)
제목: 노태악 선관위원장 "국민 모두 선거 결과에 승복해야"
내용: 노태악 선거관리위원회 위원장이 대선을 앞두고 국민 모두가 선거 결과에 승복해야 한다고 말했다.

분류: 1 (중립)
판단 근거:
- 행위: "말했다" → 단순 발언 인용
- 서술: 평서형, 감정 표현 없음
- 전체: 공식 발언만 전달

### 예시 5: 비판 (2)
제목: 김용태 "이재명, 대법원을 정치 하청기관으로 여겨 사법농단"

분류: 2 (비판)
판단 근거:
- 행위: "사법농단" → 명확한 부정적 행위
- 서술: "하청기관으로 여겨" → 비판적 표현
- 전체: 이재명 대통령에 대한 강한 비판

### 예시 6: 비판 (2)

분류: 2 (비판)
```

## 주요 개선사항 (v1 → v2)
