# 빠른 검수 체크리스트

## 📝 1분 판단 가이드

### 옹호 (0) 체크리스트
- [ ] 긍정적 형용사가 있는가? (성공적, 효과적, 훌륭한)
- [ ] 업적/성과를 강조하는가?
- [ ] 긍정적 인용문 중심인가?
- [ ] 호의적인 프레이밍인가?

→ **3개 이상 Yes면 옹호(0)**

---

### 중립 (1) 체크리스트
- [ ] 육하원칙만 전달하는가?
- [ ] 감정적 표현이 없는가?
- [ ] 찬반 균형있게 제시하는가?
- [ ] 기자 의견이 없는가?

→ **3개 이상 Yes면 중립(1)**

---

### 비판 (2) 체크리스트
- [ ] 부정적 표현이 있는가? (논란, 비난, 문제)
- [ ] 문제점을 강조하는가?
- [ ] 비판적 인용문 중심인가?
- [ ] 의혹/스캔들을 부각하는가?

→ **3개 이상 Yes면 비판(2)**

---

## 🚨 애매할 때 판단 순서

1. **제목의 톤** 확인 (긍정/중립/부정)
2. **첫 문장** 확인 (리드 문장이 핵심)
3. **인용문** 확인 (누구 말이 더 많은가?)
4. **전체 느낌** 신뢰 (첫 인상이 중요)
5. **여전히 애매하면** → **중립(1)**

## ⚡ 빠른 키워드 매칭

### 옹호 신호 키워드
```
✅ 성공, 효과, 호평, 환영, 지지, 박수, 높이 평가
   성과, 업적, 훌륭, 탁월, 긍정, 기대
```

### 중립 신호 키워드
```
⚪ 밝혔다, 전했다, 말했다, 발표했다, 개최했다
   참석했다, 방문했다, 만났다
```

### 비판 신호 키워드
```
❌ 논란, 비난, 지적, 우려, 반발, 의혹, 문제
   실패, 하락, 저조, 부진, 비판, 맹비난
```

## 🎯 3초 판단법

### Step 1: 제목 보고 첫 느낌 (긍정/중립/부정)
### Step 2: 내용 훑어보며 확인
### Step 3: 느낌과 일치하면 확정, 아니면 다시 읽기

## 💡 자주 틀리는 케이스

### ❌ 잘못된 판단
```
제목: "○○○ 후보, 시민들 만나"
판단: 중립(1)
```
**맞음!** 사실만 전달

### ❌ 놓치기 쉬운 비판
```
제목: "△△△ 후보, '이번에도' 공약 발표"
판단: 중립(1)?
```
**틀림!** "이번에도" → 부정적 뉘앙스 → **비판(2)**

### ❌ 숨은 옹호
```
제목: "◇◇◇ 후보, 시민들과 따뜻한 소통"
판단: 중립(1)?
```
**틀림!** "따뜻한" → 긍정적 평가 → **옹호(0)**
